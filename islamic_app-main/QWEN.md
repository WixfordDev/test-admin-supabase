- DeenHub is a comprehensive Islamic Flutter app with the following key characteristics:

1. Project Structure:
- Feature-based architecture with separate directories for each feature (quran, hadith, prayers, etc.)
- Clean architecture with data/domain/presentation layers
- Uses BLoC for state management
- Uses GoRouter for navigation with bottom tab navigation
- Uses Supabase for backend services including real-time multiplayer trivia, authentication, and data storage

2. Key Features:
- Quran reading with memorization feature
- Hadith collection with search functionality
- Prayer times with calculation methods
- Qibla compass
- Zakat calculator
- AI Chatbot for Islamic guidance
- Duas collection
- Trivia game (solo and group) with real-time multiplayer
- Nearby mosques with map integration
- Prayer guides (Hajj, Wudu)
- Subscription model with in-app purchases

3. Technical Implementation:
- Uses Flutter with Material 3 design
- BLoC pattern for state management
- GoRouter for navigation with deep linking
- Supabase for backend services (auth, real-time, storage)
- Drift for local database
- Firebase for analytics, crashlytics, and messaging
- Multi-language support with easy_localization
- Modular architecture with dependency injection (get_it)

4. Navigation:
- Bottom navigation with Home, Mosque, Prayers, Quran, and More tabs
- More tab implemented as a bottom sheet
- Tab-based navigation with stateful shell routes
- Deep linking support for various screens

5. Backend Integration:
- Supabase for real-time multiplayer trivia
- Authentication with email, Google, and Apple sign-in
- Real-time features for trivia games
- User profiles and subscription management
- Memorization progress synchronization
