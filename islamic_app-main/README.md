# deenhub

The app's design is nice and modern looking. 
The app will mave Zakat, AI Chatbot in More options of bottom navigiation.


## Features

### Dashboard


### Quran

- Memoraization Feature

### Hadith Collection

- Search Hadith by number, book, or chapter name.
- Show hadiths from json

### Qibla Compass

- Feature will be exactly like this website "https://qiblafinder.withgoogle.com/intl/en/finder/ar"
- Shows user camera screen with a qibla compass
- Accurate Qibla compass to determine the prayer direction.


### Zakat Calculator

- Shows user zakat calculation

### AI Chatbot

- Intelligent assistant for answering Islamic queries and guidance.
- It uses Chatgpt as an AI Chatbot which will answer only Islamic questions


### Daily Islamic Reminders

-  Receive notifications for daily spiritual motivation and important Islamic events.

### Dua Collection

- Access a library of authentic Duas for various occasions.
